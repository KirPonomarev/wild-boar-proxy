## Canon Decisions

- Keep the screen a registry, not a config editor.
- Show only secret references and reference status; never show or accept secret values.
- Keep row mutations deferred in this contour. The only active row action may be `api_route_validate` when route and command metadata allow it.
- Represent create/update as `deferred` until a bounded server-side route builder exists.
- Live failure must not reuse fixture data as runtime truth.
